# Acknowledgements

Text Scramble is an original pitch.dog project built on the public extension platforms provided by Figma and Adobe.

The first Raycast version of this workflow was informed by [Change Case for Raycast](https://www.raycast.com/erics118/change-case) by erics118. No Change Case source code is included here; credit is given for the interaction pattern that helped spark the original tool.

Development uses the official Figma Plugin API, Adobe InDesign UXP and ExtendScript APIs, TypeScript, esbuild, and the Node.js test runner. Their respective projects and licenses remain with their authors.
